# Advent of Code 2022 (Rust)

## Generate a day

```bash
cargo run --bin generator <day>
```

Example:
```bash
cargo run --bin generator 1
```

## Run a Solution

Test input (default):
```bash
cargo run --bin aoc <day>
```

Main input:
```bash
cargo run --bin aoc <day> main
```

## Where to code

Edit only:
```
src/days/dayXX/solution.rs
```

Inputs:
```
input_test.txt
input_main.txt
```

All boilerplate and wiring are auto-generated.
